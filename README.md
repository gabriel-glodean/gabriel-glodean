## 🚀 Projects

### 🔍 [Constant Tracker](https://github.com/GlodeanGabriel/constant_tracker)
A Spring Boot 3 / WebFlux service that extracts and indexes Java class constants from compiled bytecode.
- Custom bytecode analyser using ASM opcodes
- Solr-backed full-text search with parent/child document model
- Redis atomic versioning and Spring Cache integration
- 85%+ JaCoCo coverage enforced on all modules
- Fully containerised (Docker Compose + Terraform)

### 🎨 [Model Painting Scheme Helper](https://github.com/gabriel-glodean/manual_paint_scheme_color)
A tool to help hobbyists manage and track painting schemes for miniature models.

Home-lab gateway for Raspberry Pi — scan your LAN, wake devices via Shelly plugs, and proxy services by MAC address.
